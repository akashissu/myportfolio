# PAP-463: Create a modern, premium, highly interactive personal portfolio website for a Full Stack AI Engineer.

Create a modern, premium, highly interactive personal portfolio website for a Full Stack AI Engineer.

The portfolio should feel futuristic, professional, and elegant, combining clean SaaS-style UI with AI/technology-inspired visuals. The design should be visually impressive but still easy to read, fast, responsive, and user-friendly.

Website owner role:
Full Stack AI Engineer

Main goal:
Showcase my skills, projects, AI expertise, full-stack development experience, and professional profile in a way that helps me attract clients, recruiters, startups, and tech companies.

Overall Design Style:
Use a modern dark theme with glassmorphism, gradients, soft glowing effects, animated backgrounds, and smooth micro-interactions. The website should look premium, advanced, and suitable for an AI engineer.

Color Palette:

* Background: deep navy, black, or dark charcoal
* Primary accent: electric blue, cyan, violet, or neon purple
* Secondary accent: soft green or teal
* Text: white and light gray
* Cards: semi-transparent dark glass panels
* Buttons: gradient background with glow hover effect

Typography:
Use a clean modern font such as Inter, Satoshi, Manrope, or Space Grotesk.
Headings should be bold, large, and futuristic.
Body text should be readable and clean.

Page Sections:

1. Navbar
   Create a sticky transparent glass navbar with blur effect.
   Include these menu links:

* Home
* About
* Skills
* Projects
* Services
* Experience
* Testimonials
* Contact

Navbar requirements:

* Smooth scrolling to each section
* Active section highlight
* Mobile hamburger menu
* Subtle hover animation
* Resume download button on the right
